What is Vue.js?

A progressive framework

  • Standalone core functionalities
  • Official support for common libraries
  • Optional Plugins and Build Tools
  • Progressive in a sense that you can use the framework alone to build a whole single page app
  • The core functionalities does not need other dependencies unlike other frameworks that rely on jQuery DOM utilities
  • Libraries for state management, SPA routing, data fetching
    • Vuex
    • Vue Router
    • Vue Resource
  • You can customize how you build your code with Vue

Life Cycle Hooks
  1. beforeCreate
  2. created
  3. beforeMount
  4. mounted
  5. beforeUpdate
  6. updated
  7. beforeDestroy
  1. beforeCreate - Called synchronously immediately after the instance has been initialized, before data observation and event/watcher setup.
  2. created - Called synchronously after the instance is created. At this stage, the instance has finished processing the options which means the following have been set up: data observation, computed properties, methods, watch/event callbacks. However, the mounting phase has not been started, and the $el property will not be available yet.
  3. beforeMount - Called right before calling render function for mounting the component
  4. mounted - Called after the instance has been mounted to an element. Does not guarantee that all child components have been mounted
  5. beforeUpdate - Called when a data is changed and before applying the patch to the DOM
  6. updated - Called after applying the patch to the DOM
  7. beforeDestroy - Called before the instance is destroyed
  8. destroyed - Called after the instance has been destroyed. No longer have access to the instance.

Testing





// testing
import Vue from 'vue'
import MyComponent from './MyComponent.vue'

// helper function that mounts and returns the rendered text
function getRenderedText (Component, propsData) {
    const Constructor = Vue.extend(Component)
    const vm = new Constructor({ propsData: propsData }).$mount()
    return vm.$el.textContent
}

describe('MyComponent', () => {
  it('renders correctly with different props', function () {
      expect(getRenderedText(MyComponent, {
          msg: 'Hello'
      })).toBe('Hello')

      expect(getRenderedText(MyComponent, {
          msg: 'Bye'
      })).toBe('Bye')
  })
})

import { mount } from '@vue/test-utils'
import MyComponent from './MyComponent.vue'

it('renders correctly with different props', function () {
    // Now mount the component and you have the wrapper
    const wrapper = mount(Counter, {
        propsData: {
            msg: 'Bye',
        }
    })

    // You can access the actual Vue instance via `wrapper.vm`
    expect(wrapper.vm.text()).toBe('Bye')
    wrapper.vm.setProps({ msg: 'Haro' })
    wrapper.vm.update()
    expect(wrapper.vm.text()).toBe('Haro')
})
